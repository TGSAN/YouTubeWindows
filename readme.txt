YouTube Windows

要求

操作系统：

Windows 7/8.1/10/11
Windows Server 2008 R2/2012 R2/2012/2016/2019/2022

处理器：

x86/x64/arm64

运行时：

1.	Microsoft Edge (Chromium) 非稳定版本 86.0.616.0 及以上
	或 WebView2 Runtime：https://developer.microsoft.com/zh-cn/microsoft-edge/webview2/
	或将 WebView2 Shared Runtime 放入 runtime 目录（PlayReady DRM 将不可用）

2.	.Net 4.5.2+

高级

命令行参数：

--allow-endscreen				允许显示片尾“视频内链接”叠加层
--enable-features={Feature1,Feature2,...}		使用Feature1,Feature2,...等特性（参考 Microsoft Edge 文档）
